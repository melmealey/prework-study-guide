HTML

  • HyperText Markup Language:STRUCTURE
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element is a type of content on a webpage: text, image, header, list. There are also Meta-related elements, Content division elements and Semantic elements
  • HTML attributes are always defined in the beginning element, never the closing elements.
  • The head element contains information about the webpage. Important for the browser, not the user.
  • The body element represents the visible content shown to the user. Parent element
  • Elements that are nested inside other elements are called child elements.
  • If there are 3 elements nested inside of the body element, those 3 are known as child elements.
  • Order Matters!

CSS

  • Cascading Style Sheets
  • Let's add style!
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

  • External CSS Style sheets-MOST COMMON WAY
  • Declarations=Property and Value ie:Color:blue
